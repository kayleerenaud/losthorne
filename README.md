# Losthorne: Last Light 🏮

A two-player co-op medieval adventure game for iPhone (web app), designed by **Kaylee, Josiah & Malachi** and built with Claude.

> All the land has been conquered — except one village. You are its last warriors.

## The game
- **Top-down Zelda-style** adventure, played in landscape on a phone
- **Two-thumb controls**: left thumb moves, right thumb fights
- **Combat**: swipe to slice, hold to SMASH, or draw the bow and pull back to shoot
- **Quests** from Chief Bonbottom, with rewards and free exploration between
- **Random-power potions** from Dorgan (strength / speed / stoneskin)
- **Hearts & hunger**: eat blueberries… never the red ones
- Room-code co-op (currently mocked — real multiplayer coming after the story is built out)

## Play it
Open `prototype/index.html` in a browser — everything is one self-contained file (no build step, no server).
